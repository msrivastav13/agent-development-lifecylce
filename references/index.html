CLI Quick Reference

Build

sf agent generate authoring-bundle Scaffold a new .agent + bundle-meta.xml
sf agent validate authoring-bundle Local syntax and structure validation
sf agent publish authoring-bundle Create runtime entities from the bundle
sf agent activate Make the published agent live
sf agent deactivate Take an active agent offline

Test

sf agent preview start/send/end Interactive preview sessions with traces
sf agent test create Deploy a test definition from YAML spec
sf agent test run Execute a test suite against an agent
sf agent test results Retrieve async test results
sf agent test run-eval Run multi-turn evaluation tests
sf agent generate test-spec Generate YAML from existing test definition

Navigate

sf org open authoring-bundle Open Agentforce Studio in browser
sf org open agent Open a published agent in browser
